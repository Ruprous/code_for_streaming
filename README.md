# code_for_streaming

このリポジトリは、OBSなどの配信ツールを活用する際に便利なPythonスクリプトをかき集めたディレクトリ
配信のクオリティ向上や作業効率化、集計、一部自動化を目的としたさまざまなツールが含まれていく予定
自分用に作ったものを適当に入れてるため、ターミナルやcmdでのinstallが必要なコードもあります
ご自由に変更してご使用ください
